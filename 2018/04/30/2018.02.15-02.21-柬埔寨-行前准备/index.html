2018.02.15-02.21 | 柬埔寨·行前准备

发表于 2018-04-30 分类于 旅游那些事儿

　　因为这次出行前买机票、订酒店、搞签证、订演出票 blablabla 的事情太多了，于是决定写一个依旧充满流水账气息的纯攻略篇记录一下我的那些辛酸泪……

PART I · 机票

　　出国旅行最先搞定的一般是机票，只要你有护照了就能买了（如果你连护照都还没有…那还是先办护照吧）…没有什么特别的原因，只是因为机票越早买越便宜啊筒子们！！！（来自穷人内心深处的哀嚎

　　只要你不是去什么奇怪的地方或者是什么特殊的身份导致签证比较难办下来，一般都不会出现买了机票签证又下不来最后自己把自己坑了这种情况的…万一你真的这么把自己坑了这个锅我也不背…当然还有一种情况可能是预留办签证的时间不够导致签证没来得及办下来…这还是你自己的锅

　　去 柬埔寨 要么飞 金边 、要么飞 暹粒 ……目前 深圳 到 金边 和 暹粒 都有往返直飞的航班（南航和深航都有，一天一班的那种，其他航空公司的我没查）……当然坐飞机出国自然就会想到 香港 ，香港 直飞这两个地方的航班相对多很多选择……然而从 香港 飞并没有比从 深圳 飞要便宜多少（不仅不便宜反而还更贵），同时综合考虑便利性的因素，最后决定直接从 深圳 飞…

　　决定好出发地之后就要决定目的地…

　　飞 金边 的机票比飞 暹粒 的机票便宜很多…然！而！ 暹粒 是一定会去的一个地方…于是穷人本人我在各种查询罗列比较了：

  1. 深圳 <-> 暹粒 往返
  2. 深圳 -> 暹粒、暹粒 -> 金边、金边 -> 深圳
  3. 深圳 -> 金边、金边 -> 暹粒、暹粒 -> 深圳
  4. 深圳 <-> 金边 往返、金边 <-> 暹粒 往返

　　这几种方案的总价，并结合我们准备出行的天数、时间之后，最终决定采用最后一种方案，也就是需要买 深圳 - 金边 往返的国际机票 && 金边 - 暹粒 往返的 柬埔寨 国内机票

　　深圳 - 金边 往返的国际机票我是 2017.12.25 南航会员日的时候在官网买的…

深圳-金边 往返机票总价：￥2579 * 2 + ￥2739 = ￥7897

这里要说明一下…如果要买不止一张机票的话，比如我们家一共要买三张，在人数选 “3” 查询完价格后，请一定！一定要多手顺便查查看人数为 “1” 时的报价，看乘以 3 之后是不是正好是人数选 “3” 时的报价…… #强迫症的自我修养
因为啊…我就是手多多了这么一下，发现人数 “1” 的报价乘以 3 并不等于人数 “3” 的报价呢！（准确来说是比人数 “3” 的报价要便宜）
进而发现了，往返 ￥2579 价格舱位的票只剩两张了，如果人数选 “3” 的话，三张票都会以 ￥2739 的价格计算，也就是 ￥2739 * 3 = ￥8217
所以最后我是先买了两张 ￥2579 的票，再另外买一张 ￥2739 的票
（有时候啊…手多多一下没坏处的…你看我这一下省了 ￥320 呢！）

　　接下来该买 金边 - 暹粒 往返的 柬埔寨 国内机票了（如果是我自己，估计会选择坐大巴，毕竟过着吃土的生活…奈何大巴这种交通方式父上大人和母上大人必定是瞧不起的…唉）

　　我是先在携程上查有哪些航空公司的航班，然后再去那些航空公司的官网查价格买票的…（毕竟曾经被携程伤害过，轻易不敢再在携程上买机票了…尽管它上面的价格比官网要便宜）

　　最后我选的是澜湄航空，当然还是因为…便宜啊！！！

　　澜湄航空是一家中资的航空公司，2017年9月才开始运营，感觉它的定位就是 柬埔寨 国内的廉价航空…没有飞机餐（不过飞行时间不到一个小时，没有也无所谓了），每人只能托运一件行李，而且还限重的那种…（廉价航空有一点比较不好的就是…飞机延误可能会比较严重…如果对飞机延误接受度较低的话请慎重考虑…）

　　抱持着从南航订票学习到的宝贵经验，我这次也是人数 “3” 的价格查完后顺手查了一下人数 “1” 的价格，乘以 3 后发现两者是一样的……Mmm…那就直接买三张就好啦…（其实机票本身不贵，然而燃油附加费几乎是机票价钱的两倍……）

金边-暹粒 往返机票总价：USD$79.5 * 3 = USD$238.5（换算成人民币约 ￥1569.78）

　　至此，所有机票购买完成~

机票总费用：￥7897 + ￥1569.78 = ￥9466.78，平均 ￥3155.6/人

PART II · 住宿

　　和机票一样，只要时间定下来就能提早订的还有住宿…越早订可能越便宜是一方面，还有一方面是…太晚订的话位置好又性价比高的酒店会没房啊筒子们！！！

　　订国外的住宿最好有一张 VISA 或 MasterCard 的信用卡，虽然现在越来越多地方支持银联，但 VISA 和 MasterCard 还是主流，备一张比较好…

　　住宿我习惯在 Booking 订（因为感觉他们家的用户交互和 UI 做得相对较好），也没有太多去查别的网站（主要因为懒）…如果有时间不嫌麻烦的话还是建议多查几家网站对比一下的……

　　在 Booking 订的话一般需要使用上述的两种信用卡预订，房费入住的时候再交，但是会冻结信用卡的一定金额作担保…

　　其他订住宿的东西就不多说了，这里主要说一下 Booking 有一个 “价格更放心” 的政策，如果发现相同日期、相同房型在其他网站能以更低的价格预订，那么在入住日 24 小时之前可以申请差价补偿…

　　（是的没错…这又是我手多多的结果……）

　　在 Booking 订好了 金边 和 暹粒 住宿之后的某一天…也许是命运石之门的选择…我去 Booking 的兄弟网站 agoda 查了一下同一家酒店相同日期、相同房型的价格，发现…怎么便宜这么多！然后依稀想起来 Booking 好像可以申请差价补偿，于是就去申请了…

　　总的来说申请差价补偿不算复杂，直接在订单页面就可以找到申请差价补偿的入口，然后会让你填写其他网站的价格等等信息，并附上链接即可…等待几个工作日就会收到 Booking 的邮件答复你申请结果……如果不想等这么久的话，也可以打 Booking 的 24 小时客服电话进行申请，这样时效性会比较好，可以用你实时看到的价格进行申请…

　　毕竟我是一个比较 shy 的 girl…我是在网上申请的，需要等几个工作日才收到回复，在这几个工作日的时间里，其他网站的价格可能会变动、或者相同日期内相同房型可能会被订完进而无法再查询到价格…而我，大约又是命运石之门的选择，这两种情况都遇到了……

情况 A：其他网站仍可在相同日期内预订相同房型，但价格较申请时相比有变动

　　几个工作日后会收到订单信息更新的邮件，通知你订单价格已经更改…同时会收到另一封邮件说明此价格是如何计算得出的…（会基于他们处理申请时对方网站的价格进行计算

　　这种情况比较简单，坦然地接受就好了

情况 B：处理申请时，其他网站相同日期内的相同房型已被订完，无法再查询到价格

　　遇到这种情况，几个工作日后也会受到 Booking 的反馈邮件，告诉你他们无法核实相关价格…但在回复我的邮件中，他们写道：“但为了向您表示感谢，我们愿为您提供金额为您提出的差价50%的退款”…不知道这是不是统一的政策？我没仔细查过……总之，就还是能收到补偿的…

　　不过，要想拿到这个补偿，就有点麻烦了……

  • 首先，需要保留好入住的发票或账单，并在入住完成后，将发票或账本副本发送至指定邮箱
  • Booking 核实申请后会让你提供更详细的账单明细信息
  • Booking 核实账单信息后，会邮件发给你一个链接，点击链接并提供希望接收退款的银行卡信息，然后等着收钱就好了

　　这里说明一点，最后接收退款的银行卡可以不是当初付款的银行卡，甚至不是本人的银行卡也行…而且退款退的是人民币，但是银行卡必须是 VISA 或 MasterCard …（这让单币卡情何以堪…

　　以上，就是关于住宿要说的啦~

PART III · 签证

　　柬埔寨 的签证大致可以分为三种：纸质签证、电子签证、落地签

　　电子签证最便宜，不过纸质签证比较…嗯…有情怀……

　　作为一个收集癖，我是很想要纸质签证的，但是纸质签证要么要找旅行社（会有中间商赚差价），要么要去大使馆（太麻烦了也没时间）…最后，还是忍痛选择了申请电子签证…（说到底还是因为穷…

　　申请电子签证的攻略网上一搜一大把，我就不赘述了…主要说一些要注意的点…

　　网上申请电子签证需要使用 VISA 或 MasterCard 付款，要提前备好…

　　申请的时候会让你填一些基本资料，其中有一项是入境的方式（乘坐的交通工具），会让你填写航班号，但是申请的时候这个不是必填项，空着不写也能成功提交申请…然而你会发现…怎么迟迟收不到申请结果呢？？？所以如果已经买好机票，记得及时将航班号信息补充上去…填好之后大约 1-2 个工作日就能收到电子签证了（超！快！的！

　　电子签证需要打印至少两份带着，一份入境的时候给，一份出境的时候给，亲测打印黑白的也没问题…

　　另外就是填写入境卡和出境卡…在飞机上，大概还有半个小时就降落的时候，空姐会发出入境卡给乘客填写（可以要多几张以防万一写错了）…

　　所以最好带一支笔在身上（说不定还能借笔认识小哥哥小姐姐呢）…填写的时候如果有不确定的地方可以先空着，问了空姐之后再写…好像一张出入境卡上最多可以有三处修改……因为是飞机快要降落的时候才发，降落之前又会被要求收起小桌板，所以总共让你填写的时间可能就十几分钟…对于我这种写字慢的人来说时间可能是有点不够的……其实写不完也没关系，因为下飞机之后入境之前，是会有地方有桌子让你写的，所以完全不需要在飞机上写完…

　　还有就是，出入境卡上会有一栏要让你填写在 柬埔寨 国内的联系地址，入境的时候会被要求这一栏一定要写，所以可以直接把预订的酒店地址写上去…

　　总的来说，申请电子签证和填写出入境卡都不难，一步步跟着做基本都没什么太大的问题的…

电子签证申请费用（含手续费）：USD$36 * 3 = USD$108

PART IV · 《吴哥的微笑》演出门票

　　事情是这样的…母上大人说想看，查了一下好像口碑还不错，于是就去买票了……（完）

　　习惯性的上官网去查查演出信息，场次时间、地点什么的，然后看到官网可以预订门票，似乎还有优惠，于是就在官网上订了…

　　在官网上订票就真的只是[订]票，不需要支付任何费用…填写完订票信息后，会收到邮件跟你确认信息和预订的场次…

　　不过这些都不是重点，我之所以把这一项写在这里的重点是……我，在网上，用购买 B 区票的价格，买到了 A 区的票！（A 区是中间，相当于 VIP 位，B 区是外围）…这似乎是在官网订票的优惠~

　　如果想看这个演出，不妨先上官网看看，也可以查查一些攻略对比一下价格…

PART V · 电话卡

　　各种运营商的价格对比和推荐网上一搜一大堆，我也没有都用过也就不推荐什么了，这里主要说一下我的使用体验吧…

　　电话卡我并不是提前买好，而是去到下飞机才买的…（因为购买的人不多，半个钟之内就搞定了）…放在这里一起说一下，游记里面就不另外说了…

　　我去 柬埔寨 8 天用的是 seatel （东南亚电信），因为说他家是纯 4G 网络……价格上和其他运营商也差不太多，就不过分纠结那几分美金了…（而且他们的网站有中文…查阅起来非常友好（信我，真的不是因为这个原因才选它的

　　我租的是 Mifi，租用时间从租的那个时间点开始到第二天的这个时间点算一天，而不是过了 24 点就算一天，这个我觉得挺好…这样我 8 天用一个 7 日的套餐就刚刚好了~

　　虽然说是租 Mifi，但其实就是给你一部电话（Android 系统）用来发热点给自己手机用…我们家三个人租一部，每天一拖三……我本来以为这么开热点，这部机的电量会血崩，还担心移动电源会不会不够用…而事实上这部机子的电还挺耐用的…平均一天只需要插一次移动电源…

　　信号方面，我觉得 OK，除了在 吴哥窟 的某些地方信号没这么好，大部分时间信号都是很 OK 的…

　　这只是我的个人使用体验，仅供参考…

　　以上，差不多就是行前需要准备的东西了…另外有时间的话，攻略多看看没坏处的，至少了解更多行情可以帮你少点被坑钱……
